# python-practice

## intitial setup
create venv
`python3 -m venv .venv`

activate in terminal
`source .venv/bin/activate`

install flask
`pip install Flask`

run server
`flask --app api run`

practice API was built utilizing [this](https://programminghistorian.org/en/lessons/creating-apis-with-python-and-flask) guide.
